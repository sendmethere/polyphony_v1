# public/ — 정적 파일

여기에 둔 파일은 사이트 루트(`/`)에서 그대로 제공됩니다.

## 사용 방법 페이지 이미지
`/help` 페이지의 그림 자리표시자에 대응하는 이미지를 여기에 넣으세요:

- `help_image_1.png` — 첫 화면 역할 선택(교사/학생)
- `help_image_2.png` — 관리자: 첫 질문 작성·유형 선택
- `help_image_3.png` — 학생: 응답 작성칸(척도형 점수 버튼)
- `help_image_4.png` — 군집안·브리핑 검토, 군집화 입자
- `help_image_5.png` — 심화/전환/통합 후속 질문 선택
- `help_image_6.png` — 가로 스크롤 캔버스 전체
- `help_image_7.png` — 교사 폴리포니 미터(다양성·엔트로피·일치도, 색상 박스)
- `help_image_8.png` — 척도형 점수 분포 수직선
- `help_image_9.png` — 페이즈 바 + 진행 헬퍼
- `help_image_10.png` — 세션 회고(질문별 돌아보기)

파일을 넣으면 페이지에 자동으로 표시되고, 없으면 "이미지 N 자리" 박스 + 캡션만 보입니다.
